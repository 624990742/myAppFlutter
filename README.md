# myAppFlutter
### 一个对Flutter技术探索与研究的自学项目。
> 里边会记录一些自己研究和学习的痕迹。
> 分享给爱学习的你们，希望我们可以一起成长！
#### 目前实现的一些研究界面有
- 首页
- 短视频
- 商品列表瀑布流样式
- 个人信息
- 商品详情页
